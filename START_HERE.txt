🚀 БЫСТРЫЙ СТАРТ:

1️⃣  Установи зависимости:
    pip install -r requirements.txt

2️⃣  Настрой config.py:
    - Открой config.py
    - Добавь BOT_TOKEN (получи у @BotFather в Telegram)
    - Добавь CHAT_ID (получи у @userinfobot в Telegram)
    - Опционально: OPENAI_API_KEY для AI-анализа

3️⃣  Запусти AI-мозг:
    python ai_brain.py

4️⃣  Открой Telegram:
    - Найди своего бота
    - Отправь /start
    - Жми кнопку "🚀 Запустить тесты"


📁 СТРУКТУРА ПРОЕКТА:

tests/          - Базовые тесты (всегда работают)
my_tests/       - ТВОИ тесты для сайта (пиши здесь!)
ai_brain.py     - Главный файл (запускай его)
core.py         - Ядро системы
config.py       - Настройки


✍️ КАК ПИСАТЬ ТЕСТЫ:

1. Открой папку my_tests/
2. Создай файл test_*.py
3. Пиши функции test_*()
4. Запускай через Telegram!

Пример:

        r = requests.get("https://мой-сайт.com")

❓ ПРОБЛЕМЫ?

"No module named..." 
→ pip install -r requirements.txt

"Token is invalid" 
→ Проверь BOT_TOKEN в config.py

"pytest not found" 
→ pip install pytest pytest-json-report

Бот не реагирует 
→ Проверь что ai_brain.py запущен
→ Отправь боту /start
